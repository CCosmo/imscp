Welcome i-MSCP.

Internet Multi Server Control Panel (i-MSCP) is a project founded to build an impressive and powerful opensource Multi Server Control panel for personal and professional usage. It must allow to manage any shared hosting environment in an easy way without any human intervention on the server file system.
